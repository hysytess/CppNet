# CppNet

## 介绍
> 这是一个简易跨平台TCP/UDP服务器程序，仓库会同步每个版本的迭代进度。

- EasyTcpClientGO、1.0-1.2 版本：简易 winsocket 服务器程序.

- 1.3-1.5 版本：简易 select 服务器程序.
